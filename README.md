## Setup project
1. Clone the project: ``` git clone https://github.com/Damian-Czarnota/garazuj-to.git ```
2. Go to the folder when project exists: ``` cd web-app ```
3. Run ``` npm install ```
4. Run ``` npm start ```
5. Open your [localhost](localhost:3000)
